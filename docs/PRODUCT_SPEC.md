# ExplainLens Product Specification

## Product Identity

- **Name**: ExplainLens
- **Tagline**: Turn papers and complex texts into visual explainer cards and cartoon storyboards.
- **Chinese Name**: 图解复杂内容的 AI 教学导演
- **Category**: Educational AI Tool / Content Visualization

## Core Value Proposition

For readers and learners struggling with complex texts (papers, technical docs,
long-form articles), ExplainLens acts as an AI teaching director that:

1. Reads and understands the content
2. Breaks it into a pedagogically sound sequence
3. Creates visual metaphors and cartoon storyboards
4. Produces ready-to-share explainer cards

## Target Users

- Students reading academic papers
- Developers learning new technologies from docs
- Content creators making educational materials
- Researchers communicating their work to broader audiences

## Key Experience

**Input**: A block of complex text or an entire paper
**Output**: 8 visual explainer cards, each containing:
- Title
- Plain-language explanation
- Visual metaphor / cartoon scene description
- Image prompt for future AI image generation
- Key takeaway
- Source excerpt traceability

## MVP Scope (Phase 1)

- Text/Markdown input only
- Rule-based / heuristic analysis (no LLM)
- SVG placeholders (no real image generation)
- CLI interface
- JSON, Markdown, HTML output
- Offline-only operation

## Future Vision

- Real-time LLM-powered deep analysis
- AI-generated cartoon illustrations
- Interactive web-based editor
- Video/slideshow export
- Collaborative sharing
