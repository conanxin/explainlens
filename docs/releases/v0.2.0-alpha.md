# ExplainLens v0.2.0-alpha

## Highlights

- Searchable PDF support with page-aware chunks.
- Source pages, source index, and clickable citations.
- Provider architecture with local and experimental providers.
- Local HTTP provider with loopback-only safeguards.
- OpenAI provider draft with explicit external API opt-in.

## Added since v0.1.0-alpha

### PDF and source traceability
- PDF input via PyMuPDF.
- `source_pages.json`.
- `source_index.json`.
- HTML clickable citations.
- Markdown Source Appendix.
- Source quality warnings.

### Provider system
- Provider interface.
- `rule-based` provider.
- `mock-llm` provider.
- `local-fixture` provider.
- `local-http` provider.
- `openai` experimental opt-in provider.
- `provider_manifest.json`.
- `provider_prompt_pack.json` with `--dump-provider-prompt`.

### Local provider UX
- `doctor` CLI command.
- `validate-endpoint` CLI command.
- Local provider config templates.
- `docs/LOCAL_PROVIDERS.md`.

### Security
- External providers fail closed by default.
- OpenAI requires `--allow-external-api` and `OPENAI_API_KEY`.
- Local HTTP provider only allows loopback endpoints.
- CI does not call external APIs.
- API keys are not written to outputs or logs.

## What is still experimental

- OpenAI provider.
- Local HTTP provider.
- LLM output quality.
- PDF table/formula/figure handling.

## What is not included

- OCR for scanned PDFs.
- Real image generation.
- Web UI.
- SaaS deployment.
- Remote custom endpoints.
- Automatic model discovery.

## Quick start

```bash
git clone https://github.com/conanxin/explainlens.git
cd explainlens
python -m venv .venv
pip install -e ".[dev]"
python scripts/create_sample_pdf.py
python -m explainlens.cli analyze --input examples/sample_paper.pdf --output outputs/pdf_demo
```

## Provider examples

```bash
python -m explainlens.cli providers
python -m explainlens.cli doctor
python -m explainlens.cli analyze --input examples/sample_article.txt --output outputs/mock_demo --provider mock-llm
python -m explainlens.cli analyze --input examples/sample_article.txt --output outputs/local_fixture_demo --provider local-fixture
python -m explainlens.cli analyze --input examples/sample_article.txt --output outputs/local_http_fixture --provider local-http --local-http-protocol fixture
```

## Release safety boundary

This alpha release is designed to run locally by default. External API use is opt-in only.
